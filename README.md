# Drawing

<b>Gradle</b>

Add following dependency to your root project build.gradle file:

<pre>
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
</pre>

Add following dependency to your app module build.gradle file:

<pre>
dependencies {
    implementation "com.folioreader:folioreader:0.5.1
}
</pre>
